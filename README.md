# Responsive Landing Page

## Description
This project is a responsive landing page built using HTML and CSS. It includes a fixed navigation header, hero section, call-to-action button, and footer with social media links. Media queries are used to ensure the layout adapts smoothly to different screen sizes.

## Features
- Fixed header with navigation links
- Responsive design using CSS media queries
- Mobile-friendly layout
- Hero section with call-to-action button
- Responsive footer with social links
- Prevents horizontal scrolling
- Responsive images support

## Technologies Used
- HTML5
- CSS3

## Responsive Design Improvements
- Added viewport meta tag
- Used media queries for screens below 768px and 480px
- Navigation menu stacks vertically on smaller screens
- Reduced font sizes for mobile devices
- Footer links stack vertically on small screens
- Fixed overflow issues using `overflow-x: hidden`
- Images scale automatically within their containers

## How to Run
1. Download or clone the project files.
2. Open the project folder in VS Code.
3. Open `index.html` in a web browser.
4. Use Chrome DevTools Device Toolbar to test responsiveness on different screen sizes.

## Files Included
- `index.html` – Structure of the webpage
- `style.css` – Styling and responsive design

## Output
The webpage adjusts automatically for desktop, tablet, and mobile devices, providing a user-friendly and responsive experience.
